## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/hong-kong-gallery.git

# Navigate to project directory
cd hong-kong-gallery

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production
```bash
# Create production build
npm run build

# Preview production build
npm run preview
```

## 🌍 Deployment

### GitHub Pages
1. Fork this repository
2. Go to Settings > Pages
3. Select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Your site will be available at `https://your-username.github.io/hong-kong-gallery`

### Custom Domain
1. Add a `CNAME` file with your domain name
2. Configure DNS settings with your domain provider
3. Enable HTTPS in repository settings

## 🔧 Customization

### Update Content
- **Text**: Edit the HTML content in `index.html`
- **Images**: Replace files in `assets/` directory
- **Styling**: Modify CSS in the assets folder
- **Colors**: Update the color scheme in CSS variables

### Add New Exhibitions
1. Add new exhibition images to `assets/` folder
2. Update the exhibition carousel HTML
3. Modify the exhibition data in JavaScript
